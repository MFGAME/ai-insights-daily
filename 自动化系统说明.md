# AI商业洞察日报 - 自动化部署系统

## 📋 系统概述

**自动化流程：**
```
每天凌晨2点 → 生成5篇新洞察 → 推送到GitHub → Vercel自动更新网站
```

**时间：** 每天 02:00 (GMT+8)
**频率：** 每天1次
**内容：** 5篇新洞察（AI工具、电商、副业、游戏、SaaS）

## 🚀 快速启动

### 方式1：Windows任务计划程序（推荐）

**步骤1：创建任务脚本**
```powershell
# 文件位置：C:\Users\Administrator\openclaw-autoclaw\agents\laicai\workspace\projects\ai-insights-daily\scripts\auto-update.ps1

cd C:\Users\Administrator\.openclaw-autoclaw\agents\laicai\workspace\projects\ai-insights-daily

# 生成新洞察
node scripts/generate-insights.js

# 提交到Git
git add lib/data.ts
git commit -m "📚 自动更新每日洞察 - $(Get-Date -Format 'yyyy-MM-dd')"
git push

Write-Host "✅ 每日洞察更新完成！$(Get-Date)" -ForegroundColor Green
```

**步骤2：配置Windows任务计划**
1. 打开"任务计划程序"（Win+R，输入 `taskschd.msc`）
2. 点击"创建基本任务"
3. 名称：`AI商业洞察日报 - 自动更新`
4. 触发器：每天 02:00
5. 操作：启动程序
   - 程序：`powershell.exe`
   - 参数：`-ExecutionPolicy Bypass -File "C:\Users\Administrator\.openclaw-autoclaw\agents\laicai\workspace\projects\ai-insights-daily\scripts\auto-update.ps1"`
6. 完成

**优势：**
- ✅ 完全自动化
- ✅ 后台运行
- ✅ 无需人工干预

## 🔧 维护说明

### 修改生成时间

**Windows任务计划程序：**
1. 找到任务
2. 右键 → 属性
3. 触发器标签 → 编辑
4. 修改时间

### 修改生成数量

编辑 `scripts/generate-insights.js`：
```javascript
// 第198行，修改这里
.slice(0, 3);  // 改成 .slice(0, 5) 生成5篇
```

### 暂停自动更新

**Windows任务计划程序：**
1. 找到任务
2. 右键 → 禁用

## ⚠️ 注意事项

1. **Git凭证**：确保Git已保存凭证，否则push会失败
2. **网络连接**：确保服务器能访问GitHub

## 📞 故障排查

### 问题1：任务未执行

**检查：**
- 任务计划程序中任务是否启用
- 触发器时间是否正确
- 用户权限是否足够

### 问题2：执行失败

**检查：**
- Node.js是否正确安装
- Git凭证是否有效
- 网络是否正常

### 问题3：网站未更新

**检查：**
- GitHub是否有新commit
- Vercel是否部署成功
- 清除浏览器缓存重新访问
